# API Security Risk Analysis (Task 3)

## 🔍 API Tested
JSONPlaceholder API  
https://jsonplaceholder.typicode.com

## 🎯 Objective
To analyze API security risks and identify vulnerabilities in public APIs.

## 🛠️ Tools Used
- Postman
- Browser DevTools

## ⚙️ Methodology
- Tested API endpoints using GET requests
- Inspected headers and responses
- Identified security risks
- Classified risk severity

## ⚠️ Key Findings
- No authentication required
- Excessive data exposure
- Lack of authorization
- No rate limiting
- Missing security headers

## 🛡️ Remediation
- Implement authentication
- Apply access control
- Limit sensitive data exposure
- Add rate limiting
- Use security headers
